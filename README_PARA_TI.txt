# 📝 INSTRUCCIONES PARA TI - SÚPER SIMPLES

## 🎯 **LO QUE USTEDES NECESITAN HACER:**

1. **Crear una carpeta** en su servidor web (donde pongan las páginas de intranet)
2. **Copiar estos 4 archivos** en esa carpeta:
   - index.html
   - styles.css
   - script.js
   - (carpeta assets si existe)

3. **Darle una dirección web** como:
   - http://intranet.empresa.com/transporte
   - O la que ustedes manejen normalmente
